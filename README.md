# Hospital Management Database

## Description
This project implements a hospital management database system using SQL. It includes schema design, normalization, and queries for real-world scenarios.

## Contents
- Database schema (SQL)
- ER Diagram
- Query outputs
- Transaction implementation

## Tools Used
- DBeaver
- SQL

## Features
- Patient and doctor management
- Consultation tracking
- Prescription and medicine handling
- Billing and payment tracking
